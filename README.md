# Betty y Luis Potscats

Aplicación Android para crear y editar podcasts (Betty y Luis Potscats).

Este repositorio contiene el esqueleto inicial del proyecto Android usando Kotlin y Jetpack Compose. El objetivo inicial (MVP) incluye:

- Pantalla de grabación básica
- Gestor de proyectos local (Room) — aún por implementar
- Reproductor con ExoPlayer — integración pendiente
- Exportación con FFmpeg — integración pendiente

Cómo abrir el proyecto

1. Clona el repositorio:
   git clone https://github.com/ricayes412-art/Betty-y-luis-podcast.git
2. Abre en Android Studio (recomiendo Chipmunk o Electric Eel) y sincroniza Gradle.

Siguiente pasos que realizaré si confirmas:
- Añadir implementación de grabación (AudioRecord), pantalla de edición con forma de onda.
- Añadir pruebas básicas y CI.
